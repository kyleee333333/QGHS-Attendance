Chemical Reactions — Redox

What is a Redox Reaction?

A redox reaction involves transfer of electrons. One species is oxidized (loses e⁻), another reduced (gains e⁻).

  • Oxidation = loss of electrons
  • Reduction = gain of electrons
  • Both always occur together
  • Examples: rusting, batteries, respiration
Na + Cl₂ → Na⁺ + Cl⁻

Oxidation Numbers

Oxidation numbers are bookkeeping charges for tracking electrons.

  • Elements: 0
  • Monatomic ions = charge
  • O usually −2, H usually +1
  • Sum = charge of compound/ion
2 Na + Cl₂ → 2 NaCl
Na: 0 → +1 (oxidized)
Cl: 0 → −1 (reduced)

Balancing Redox with Half-Reactions

  1. Write oxidation & reduction halves
  2. Balance atoms except H, O
  3. Balance O with H₂O, H with H⁺
  4. Balance charge with e⁻
  5. Multiply to equalize e⁻, then add
MnO₄⁻ + 8 H⁺ + 5 e⁻ → Mn²⁺ + 4 H₂O
Fe²⁺ → Fe³⁺ + e⁻

Worked Example

Zinc + Copper(II) ions:

Zn → Zn²⁺ + 2 e⁻ (oxidation)
Cu²⁺ + 2 e⁻ → Cu (reduction)
Zn + Cu²⁺ → Zn²⁺ + Cu

This reaction powers a simple galvanic cell.

Applications & Summary

  • Batteries: galvanic cells
  • Corrosion: rust prevention
  • Industrial: electroplating, extraction
  • Biology: respiration, photosynthesis

Summary: Redox = electron transfer, tracked by oxidation numbers, balanced via half-reactions.
